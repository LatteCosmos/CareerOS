# CareerOS 🚀

> The operating system for your Australian tech career.

A high-fidelity mobile app prototype helping international IT Master's students at Australian universities accelerate their transition into junior developer roles.

**[Live Demo →](https://career-os-pink.vercel.app/)** 

## ✨ Features

- 🎯 **Smart Elective Recommender** — matches UTS electives to your target role using live job market data
- 📄 **Portfolio Builder** — auto-generates a developer portfolio from GitHub & resume
- 🧭 **Skill Gap Scanner** — benchmarks your skills against real Australian job listings
- 👥 **Mentor Matching** — connects students with industry mentors at Atlassian, Canva, Afterpay, CommBank

## 🛠 Tech Stack

- **React 18** + **Vite** for fast dev experience
- **Tailwind CSS** for design system
- **Lucide React** for icons
- 12 interactive screens, iPhone 15 Pro dimensions (393 × 852)

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.
